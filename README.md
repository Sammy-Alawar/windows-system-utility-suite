# Windows System Utility Suite
A menu-driven toolkit that makes it easy to manage and inspect a Windows system.  
This project includes two versions — **Batch** and **PowerShell** — both offering a simple interface for performing common system, file, network, and process operations without needing to memorize long commands.

Whether you're troubleshooting, learning Windows internals, or automating repetitive tasks, this suite provides a fast and intuitive way to get things done.

---

## What You Can Do With It

### **System Information**
1. Display detailed configuration information about the computer and its operating system
2. Get basic information about the computer caption, codeset, buildnumber
3. Get information about the processor
4. Get information about the installed memory
5. Get information about disk drives
6. Display the hostname portion of the full computer name
7. Display TCP/IP network configuration information
8. Display the current date
9. Display the current time

### **File Operations**
1. Display or change file attributes
2. Display the name of or change the current directory
3. Copy one or more files to another location
4. Delete one or more files
5. Display a list of files and subdirectories in a directory
6. Create a directory
7. Move one or more files from one directory to another directory
8. Remove a directory
9. Rename a file
10. Get permission for a file
11. Set permission for a file
12. Save the current directory then change it
13. Remove subdirectories
14. Replace a file
15. Graphically display the directory structure of a drive or path
16. Copy files and directory trees

### **Network Tools**
1. View the current password & logon restrictions for the computer
2. Display current server or workgroup settings
3. View the details of a particular user account
4. Stop and start a particular service
5. Display network statistics of the workstation
6. Display information about your connections  

### **Process Management**
1. View the List of Running Processes
2. Kill a Particular Process
3. Start a New Process
4. View processes consuming the most CPU
5. View processes consuming the most memory 

### **Data Utilities**
1. Search for a string within a file
2. Search for a string within a directory
3. Sort the contents of a text file.
4. Search for a specific string in a text file
5. Compare two files and display the differences
6. Display the contents of a text file 

A simple menu interface guides you through each task, making it ideal for beginners and power users alike.
